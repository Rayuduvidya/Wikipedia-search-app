# Wikipedia Search Application

This project is a simple web application that allows users to search for articles using the Wikipedia API. The application displays search results including the title, description, and link to the full article.

## Features
- Search for Wikipedia articles
- Displays article title and short description
- Clickable links to open the full article
- Responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Wikipedia API

## How to Use
1. Open the application in a browser.
2. Enter a keyword in the search bar.
3. Press Enter or click the search button.
4. The application will display related Wikipedia articles.
